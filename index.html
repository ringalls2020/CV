Senior Design Project.

The Emotiv Epoc+ EEG headset was used to capture and send the user’s EEG signals, via Bluetooth to the Raspberry Pi for data processing. A Python program processed the signals for classification and then employed a neural network to evaluate the presence of alpha, theta and beta frequency amplitudes, indicative of meditative states. Baseline and meditative neural activity have assigned infrared signals. The lamp’s bulb is an infrared controlled LED with a Bluetooth speaker. The infrared signals create a change in the lamp’s color and brightness through a CuteDigi infrared shield for the Raspberry Pi. In addition, a specific reproducible EEG artifact, EOG signals, was used to activate the lamp’s music feature. This created a device that could be used hands free once turned on and allows the user to receive feedback to develop their meditation practice. Check out our code on Github!
